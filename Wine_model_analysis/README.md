# 🍷 Wine Data Preprocessing using Min-Max Scaling

This repository demonstrates how to preprocess the **Wine Dataset** using **Min-Max Scaling** in Python with Scikit-learn. The notebook covers the complete preprocessing workflow, including loading the dataset, splitting it into training and testing sets, applying Min-Max Scaling, and visualizing the results to understand the impact of feature scaling.

The project is beginner-friendly and provides hands-on practice with one of the most important data preprocessing techniques used in Machine Learning.

---

## 🚀 Features

- Load and explore the Wine dataset
- Perform Train-Test Split
- Apply Min-Max Scaling using Scikit-learn
- Scale numerical features
- Compare original and scaled data
- Visualize feature distributions before and after scaling
- Understand the importance of feature normalization
