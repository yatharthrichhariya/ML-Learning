# 📊 Bivariate Analysis using Python

## 📌 Overview

This repository demonstrates **Bivariate Analysis** using Python to explore the relationship between two variables through data visualization. It covers different plotting techniques that help identify trends, correlations, distributions, and patterns in datasets. The notebook uses popular Python libraries such as **Pandas**, **Matplotlib**, and **Seaborn** along with real-world datasets to explain each concept in a simple and practical way.

Whether you're starting your journey in Data Analytics, Data Science, or Machine Learning, this notebook provides a strong foundation in Exploratory Data Analysis (EDA).
