# 📊 Univariate Analysis using Python

This project demonstrates **Univariate Analysis** using Python libraries such as **Pandas**, **Matplotlib**, and **Seaborn**. It uses the Titanic dataset to analyze individual features through different visualization techniques and statistical measures.

## 📌 Project Overview

Univariate Analysis is the process of analyzing **one variable at a time** to understand its distribution, frequency, central tendency, and spread. This notebook covers both **categorical** and **numerical** feature analysis with practical examples.
