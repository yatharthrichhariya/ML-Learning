# 📂 Working with CSV using Pandas

This repository contains a Jupyter Notebook demonstrating how to efficiently read, write, and manipulate CSV files using the **Pandas** library in Python. It covers essential `read_csv()` parameters and practical examples that help beginners understand how to work with real-world datasets.

## 📌 Topics Covered

- Reading CSV files
- Opening CSV files from a URL
- Using custom separators (`sep`)
- Setting index columns (`index_col`)
- Working with headers (`header`)
- Selecting specific columns (`usecols`)
- Using the squeeze parameter
- Skipping rows and limiting rows (`skiprows`, `nrows`)
- Handling bad lines (`on_bad_lines`)
- Specifying data types (`dtype`)
- Parsing date columns
- Using converters
- Handling missing values (`na_values`)
- Reading large datasets in chunks (`chunksize`)
- Exporting DataFrames to CSV

## 🛠️ Technologies Used

- Python 3.x
- Pandas
- Jupyter Notebook

## 📂 Dataset Used

The notebook demonstrates CSV operations using datasets such as:
- Weather Dataset
- Titanic Dataset
- Netflix Dataset
- Online CSV dataset from GitHub

## 🚀 Learning Outcomes

After completing this notebook, you will be able to:

- Import CSV files from local storage and URLs.
- Customize data loading using various `read_csv()` parameters.
- Handle missing values and incorrect data efficiently.
- Optimize memory usage while loading large datasets.
- Export cleaned data back into CSV format.

## ▶️ How to Run

1. Clone this repository.
2. Install the required library:
   ```bash
   pip install pandas
   ```
3. Open the notebook in Jupyter Notebook or VS Code.
4. Run the cells sequentially to understand each CSV operation.

## 📖 Repository Purpose

This notebook is designed for beginners in **Data Analysis**, **Data Science**, and **Machine Learning** who want to build a strong foundation in working with CSV files using Pandas.
